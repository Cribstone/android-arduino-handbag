Handbag: Create Arduino-based Android accessories without writing Android code
------------------------------------------------------------------------------

Make Arduino accessories for Android without writing Android code with
the HandBag App for Android and Handbag library for Arduino.

My original code or modifications licensed under the LGPL 2.0.

Android application based on DemoKit example, see code for license details.
